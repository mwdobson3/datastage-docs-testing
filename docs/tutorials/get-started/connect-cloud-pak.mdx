# Connect to Cloud Pak for Data

⏱️ **Duration:** 7 minutes

## Overview

Now that your Remote Engine is deployed, you need to register it with Cloud Pak for Data so you can orchestrate jobs from the central control plane.

:::info Coming Soon
This tutorial is under development. Check back soon for complete instructions.
:::

## Quick Steps

1. Log in to Cloud Pak for Data
2. Navigate to **Administration** → **Remote Engines**
3. Click **Add Remote Engine**
4. Enter the engine endpoint URL
5. Verify connection
