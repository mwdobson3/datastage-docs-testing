## Prerequisites

Before starting this tutorial, ensure you have:

- **AWS Account** - With permissions to create EC2 instances
- **IBM Cloud Account** - With access to Cloud Pak for Data
- **IBM Cloud API Key** - For authentication
- **SSH Client** - Terminal access on your local machine
- **Basic Linux Knowledge** - Familiarity with command-line operations

:::tip Estimated Time
The complete tutorial series takes approximately **65 minutes** to complete.
:::

- **Easy Management**: Docker-based deployment for simple updates
